# Uploadcare::Ruby

Ruby wrapper for Uploadcare API

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'uploadcare-ruby'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install uploadcare-ruby
